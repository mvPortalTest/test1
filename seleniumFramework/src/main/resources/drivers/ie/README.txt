If you want to run your tests in IE please add "ie" to <browser.name></browser.name> in pom.xml file.

For example:

  <user.username></user.username>
  <user.password></user.password>
  <grid2.hub></grid2.hub>
  <site.url>http://d1t0152g.austin.hp.com</site.url>
  <browser.name>ie</browser.name>
  <browser.version></browser.version>
  <browser.platform>windows</browser.platform>


Download the latest version of IE Driver here: https://code.google.com/p/selenium/downloads/list
